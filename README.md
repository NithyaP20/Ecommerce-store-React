# Ecommerce Store React

A modern e-commerce web application built using React and Vite. The application provides a complete shopping experience with product browsing, cart management, checkout functionality, and an admin dashboard for inventory management.

## Features

### Customer Features

* Browse products by category
* Search products
* Add products to cart
* Update cart quantities
* Remove products from cart
* Checkout with form validation
* Order confirmation page

### Admin Features

* View product inventory
* Add new products
* Edit existing products
* Delete products
* Monitor stock levels
* Inventory dashboard statistics

## Technologies Used

* React
* JavaScript (ES6+)
* Vite
* Context API
* useReducer
* CSS-in-JS Styling

## Installation

Clone the repository:

```bash
git clone https://github.com/NithyaP20/Ecommerce-store-React.git
```

Navigate to the project:

```bash
cd Ecommerce-store-React
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
